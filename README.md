## 搜尋功能
有些學校跟學系有支援縮寫，像輸入「台大」會出現國立臺灣大學的所有校系，而輸入「外文」會出現所有含「外國語文學系」的結果，還沒有的你去提issue我有空再補。

每個關鍵字用空格分開，像如果你想要所有電機及資工的校系，可以輸入「電機 資工」，就可以了。

如果你想找台清交成的電機跟資工，當然你可以輸入「台大 清大 交大 成大 電機 資工」，但你可以輸入「頂大 電機 資工」就好，而輸入「頂大 四中 電機 資工」就會把中字輩的結果加上去。

類似的組合有
 - 頂大: 台大, 清大, 交大, 成大, 政大
 - 四大: 台大, 清大, 交大, 成大
 - 四中: 中央, 中山, 中興, 中正
 - 師北海: 台師大, 北大, 台海大
